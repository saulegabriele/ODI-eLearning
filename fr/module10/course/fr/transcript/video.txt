Évaluer l’exploitabilité de l’open data

Je suis David Tarrant formateur et chercheur ici à l’Open Data Institute 

Que sont les 5 étoiles de l’open data?

les cinq étoiles des Données Ouvertes et Liées (LOD) sont des directives qui mesurent la qualité d’intégration des données sur le Web elles examinent l’utilité technique de l’ensemble de données allant de ‘disponible sur le Web sous licence ouverte’, une étoile à ‘faisant partie d’un Web de données’, cinq étoiles 

Combien d’étoiles sont nécessaires à l’open data?

l’open data est possible à partir d’une étoile au-delà, il s’agit de différents degrés d’exploitabilité des données trois étoiles représentent le plus petit dénominateur commun de l’open data exploitable 

Pouvez-vous expliquer les trois premières étoiles?

la première étoile stipule simplement que les données doivent être disponibles sous licence ouverte, quel que soit le format par exemple un fichier .PDF sur un site Web la deuxième étoile stipule que les données doivent être disponibles sous un format largement réutilisable et structuré et lisible par une machine comme un fichier Excel par exemple pour la troisième étoile, le format largement réutilisable et structuré et lisible par une machine ne doit pas être limité à un pack logiciel en particulier le format lui-même et son implémentation doivent être ouverts permettant aux gens d’utiliser de nouveaux outils ou codes pour lire ce format 

Et les deux dernières étoiles?

les deux dernières étoiles ne sont pas toujours appropriées et les implémenter peut quelquefois compromettre l’exploitabilité en faveur de l’excellence technique
