Trouver des données cachées

Je suis David Tarrant formateur et chercheur ici à l’Open Data Institute 

Qu’entend-on par ‘données cachées’?

les données ne sont pas toujours faciles à trouver nous, humains, consommons les choses de manière visuelle on regarde de belles pages Web et les images sur ces sites nous aident à savoir où trouver et télécharger des données les ordinateurs fonctionnent différemment ils aiment la structure, la logique, la clarté et surtout le texte notre défi est de comprendre comment une machine consomme des données et intégrer sa façon de penser pour extraire des données cachées 

Où sont les ‘données cachées’?

le Web est un environnement riche en données et beaucoup de sites combinent déjà des données de différentes sources afin de vous offrir des ensembles d’informations regroupés imaginez que vous planifiez des vacances plusieurs sites de vacances regroupent déjà un grand nombre d’informations telles que des données sur les lieux, des photos, des avis, des prix et peut-être même des données en transport qui vous permettent de planifier votre trajet pour aller à un endroit précis ce que nous cherchons à faire est de réaliser que nous pouvons exploiter ces mêmes connections entre les données qui existent entre les sites, afin d’avoir nous-mêmes accès aux données et de pouvoir les présenter dans un contexte entièrement nouveau de construire un nouvel ensemble d’informations et par conséquent de nouvelles connaissances 

Comment accéder aux ‘données cachées’?

certains sites vous donnent accès aux données tout simplement en ajoutant une extension à la page que vous visitez donc si vous voyez qu’il y a potentiellement des données sur la page il se peut que vous puissiez les télécharger en ajoutant l’extension .csv Un des formats les plus courants en ligne est .JSON et vous pouvez accéder à toutes les données de gov.uk en ajoutant cette extension d’autres sites offrent des flux de données, comme des sites d’actualités que nous consommons habituellement grâce à des applications mobiles au-delà de celles-ci, les choses se compliquent un peu et il faudra peut-être rechercher les données dans le code source de la page où elles sont intégrées. ces sites les utilisent déjà donc autant les rechercher sur la page elle-même enfin, de nombreuses sources de données offrent une façon particulière pour y accéder une interface machine ou une API donc il vous faudra peut-être apprendre à les utiliser 

Quels sont les bénéfices des ‘données cachées’?

en utilisant des données faisant déjà partie intégrante d’un service existant vous garantissez une certaine pertinence et une meilleure exploitabilité de plus, vous y accédez de manière contextualisée plutôt qu’à travers une plateforme de publication distincte
